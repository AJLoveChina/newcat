# Newcat

Tomcat创建、配置。一个命令行即可，无需联网，开箱即用！

## 安装

```bash
$ npm install -g newcat
```

## 如何使用

是滴 你没看错,参数都不用~

```bash
$ tomcat
```
或者
```bash
$ newcat
```
